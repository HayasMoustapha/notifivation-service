# Notification Service

Service de notifications pour Event Planner - Gestion des emails, SMS et notifications système.

## Fonctionnalités

- Envoi d'emails transactionnels
- Envoi de SMS
- Gestion des OTP
- Notifications système
- Queue de messages

## Stack Technique

- Node.js + Express
- Nodemailer (Emails)
- Twilio (SMS)
- Redis (Queue)

## Installation

```bash
npm install
npm start
```

## Développement

```bash
npm run dev
```

## Docker

```bash
docker-compose up -d
```
